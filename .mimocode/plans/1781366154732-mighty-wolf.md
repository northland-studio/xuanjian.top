# 图片水印 + 图片查看器功能计划

## 功能概述

### 功能1：图片水印
- 用户上传图片时自动添加网站水印
- 水印使用用户提供的透明背景PNG图片
- 水印位置：右下角

### 功能2：图片查看器（Lightbox）
- 点击帖子中的图片弹出全屏查看器
- 支持缩放、左右切换、关闭
- 响应式设计，支持移动端

---

## 技术方案

### 图片水印

**依赖**：`sharp` 图像处理库

**修改文件**：
- `routes/upload.js` — 添加水印处理逻辑
- 新增 `public/images/watermark.png` — 水印图片
- `package.json` — 添加 sharp 依赖

**实现思路**：
```js
const sharp = require('sharp');

async function addWatermark(inputPath, outputPath) {
    const watermarkPath = path.join(__dirname, '../public/images/watermark.png');
    await sharp(inputPath)
        .composite([{
            input: watermarkPath,
            gravity: 'southeast',
            blend: 'over'
        }])
        .toFile(outputPath);
}
```

**流程**：
1. 用户上传图片
2. multer 保存原始文件
3. sharp 添加水印
4. 返回水印后的图片URL

### 图片查看器

**方案**：使用原生JavaScript实现（无需额外依赖）

**修改文件**：
- 新增 `public/css/lightbox.css` — 查看器样式
- 新增 `public/js/lightbox.js` — 查看器逻辑
- `public/js/common.js` — 集成查看器

**功能特性**：
- 点击图片弹出全屏遮罩
- 显示原图（非缩略图）
- 左右箭头切换图片
- ESC / 点击遮罩关闭
- 支持手势缩放（移动端）

**实现思路**：
```js
// 为帖子内容中的图片添加点击事件
function initLightbox() {
    document.querySelectorAll('.post-content img').forEach(img => {
        img.style.cursor = 'pointer';
        img.addEventListener('click', () => openLightbox(img.src));
    });
}
```

## 注意事项

### sharp 依赖
- sharp 是原生模块，需要在服务器上编译
- Docker 构建时需要安装编译工具
- 或者使用预编译的 Docker 镜像

### 水印图片
- 用户需要提供透明背景的PNG水印
- 建议尺寸：200x60px 左右
- 透明度建议：50-70%

### 性能考虑
- sharp 处理会增加上传时间（约1-2秒）
- 大图片可能需要调整处理参数
- 可以考虑异步处理或队列
