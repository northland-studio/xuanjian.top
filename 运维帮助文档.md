# 玄剑公会官网运维帮助文档

## 一、系统概述

本网站是一个基于 Node.js + Express + SQLite 的公会官网系统，部署在 Ubuntu + Nginx 环境下。

### 技术栈
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **后端**: Node.js + Express
- **数据库**: SQLite3 (本地文件存储)
- **服务器**: Nginx (反向代理)
- **操作系统**: Ubuntu + KDE桌面环境

---

## 二、部署步骤

### 1. 环境准备

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装 Node.js (使用 NodeSource)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

# 安装 Nginx
sudo apt install -y nginx

# 安装 PM2 (进程管理器)
sudo npm install -g pm2
```

### 2. 上传网站文件

```bash
# 创建网站目录
sudo mkdir -p /var/www/xuanjian

# 上传文件到该目录 (使用 scp 或 ftp)
# 确保包含以下文件和目录:
# - server.js
# - package.json
# - database.js
# - middleware/
# - routes/
# - scripts/
# - public/
# - data/
```

### 3. 安装依赖

```bash
cd /var/www/xuanjian
sudo npm install
```

### 4. 初始化数据库

```bash
# 运行数据库初始化脚本
sudo npm run init-db

# 默认管理员账号:
# 用户名: admin
# 密码: xuanjian123 (建议在部署后立即修改)
```

### 5. 配置环境变量

```bash
# 创建 .env 文件
sudo nano /var/www/xuanjian/.env

# 添加以下内容:
PORT=3000
NODE_ENV=production
JWT_SECRET=your-random-secret-key-here-change-this
ADMIN_PASSWORD=xuanjian123
```

### 6. 启动服务

```bash
# 使用 PM2 启动
sudo pm2 start server.js --name "xuanjian-website"

# 设置开机自启
sudo pm2 startup
sudo pm2 save
```

## 三、Nginx 配置

### 1. 创建配置文件

```bash
sudo nano /etc/nginx/sites-available/xuanjian
```

### 2. 添加以下配置

```nginx
server {
    listen 80;
    server_name xuanjian.top www.xuanjian.top;
    
    # 重定向到 HTTPS (如果使用 SSL)
    # return 301 https://$server_name$request_uri;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
    
    # 静态文件缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
        proxy_pass http://localhost:3000;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

## 五、故障排除

### 问题1: 网站无法访问

**检查步骤:**
1. 检查服务是否运行: `sudo pm2 status`
2. 检查端口是否监听: `sudo netstat -tlnp | grep 3000`
3. 检查 Nginx 配置: `sudo nginx -t`
4. 查看错误日志: `sudo pm2 logs`

**解决方案:**
```bash
# 如果服务停止，重新启动
sudo pm2 start server.js --name "xuanjian-website"

# 如果端口被占用，查找并结束进程
sudo lsof -i :3000
sudo kill -9 <PID>
```

### 问题2: 数据库错误

**症状:** 无法登录、无法发布内容

**检查步骤:**
```bash
# 检查数据库文件是否存在
ls -la /var/www/xuanjian/data/guild.db

# 检查文件权限
sudo chown -R www-data:www-data /var/www/xuanjian/data
sudo chmod 755 /var/www/xuanjian/data
sudo chmod 644 /var/www/xuanjian/data/guild.db
```

### 问题3: 上传文件失败

**检查步骤:**
```bash
# 检查上传目录权限
ls -la /var/www/xuanjian/data/uploads/

# 修复权限
sudo mkdir -p /var/www/xuanjian/data/uploads
sudo chown -R www-data:www-data /var/www/xuanjian/data/uploads
sudo chmod 755 /var/www/xuanjian/data/uploads
```

### 问题4: 内存不足

**症状:** 服务崩溃、响应缓慢

**解决方案:**
```bash
# 查看内存使用
free -h

# 重启服务释放内存
sudo pm2 restart xuanjian-website

# 或者设置 PM2 内存限制自动重启
sudo pm2 start server.js --name "xuanjian-website" --max-memory-restart 512M
```

### 问题5: 忘记管理员密码

**解决方案:**
```bash
cd /var/www/xuanjian
node -e "
const bcrypt = require('bcryptjs');
const sqlite3 = require('sqlite3').verbose();
const db = new sqlite3.Database('./data/guild.db');
const hash = bcrypt.hashSync('newpassword', 10);
db.run('UPDATE users SET password = ? WHERE username = ?', [hash, 'admin'], function() {
    console.log('密码已重置为: newpassword');
    db.close();
});
"
```

## 六、安全建议

1. **修改默认密码**: 部署后立即修改 admin 密码
2. **配置防火墙**: 只开放 80/443 端口
   ```bash
   sudo ufw allow 80
   sudo ufw allow 443
   sudo ufw enable
   ```
3. **定期备份**: 设置定时任务自动备份数据库
4. **更新系统**: 定期运行 `sudo apt update && sudo apt upgrade`
5. **使用 HTTPS**: 申请 SSL 证书并配置 HTTPS
